How to use this API

Download Postman Json

To connect in the API, you need authenticate with token:

Verb: POST
Method: authenticate
URL: https://dm-api-rest-with-spring-boot.herokuapp.com/api/authenticate

Get car list:

Verb: GET
Method: getCars
URL: https://dm-api-rest-with-spring-boot.herokuapp.com/api/cars

Get car list by brand:

Verb: GET
Method: getCarsByBrand
URL: https://dm-api-rest-with-spring-boot.herokuapp.com/api/cars/brand/{brand-name}

Get car by id:

Verb: GET
Method: getCar
URL: https://dm-api-rest-with-spring-boot.herokuapp.com/api/car/get/{id}

Create a new car:

Verb: POST
Method: createCar
URL: https://dm-api-rest-with-spring-boot.herokuapp.com/api/car/create

Update car:

Verb: PUT
Method: updateCar
URL: https://dm-api-rest-with-spring-boot.herokuapp.com/api/car/update/{id}

Delete a car:

Verb: DELETE
Method: deleteCar
URL: https://dm-api-rest-with-spring-boot.herokuapp.com/api/car/delete/{id}
