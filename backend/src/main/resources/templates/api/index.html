How to use this API

To connect in the API, you need authenticate with token:

Verb: POST
Method: authenticate
URL: https://dm-api-rest-with-spring-boot.herokuapp.com/api/authenticate

To get a car list:

Verb: GET
Method: getCars
URL: https://dm-api-rest-with-spring-boot.herokuapp.com/api/cars/

To get a car by id:

Verb: GET
Method: getCar
URL: https://dm-api-rest-with-spring-boot.herokuapp.com/api/cars/get/1

To create a new car:

Verb: POST
Method: createCar
URL: https://dm-api-rest-with-spring-boot.herokuapp.com/api/cars/create

To update a car:

Verb: PUT
Method: updateCar
URL: https://dm-api-rest-with-spring-boot.herokuapp.com/api/cars/update/1

To delete a car:

Verb: DELETE
Method: deleteCar
URL: https://dm-api-rest-with-spring-boot.herokuapp.com/api/cars/delete/1
